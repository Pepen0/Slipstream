# All the documentation for Slipstream

# How to view the diagrams
## in the web browser
1. go to excalidraw.com
2. upload the diagram file (eg.`docs/Diagrams/system.excalidraw`)
3. Start editing

## in Vscode
1. install the extension https://marketplace.visualstudio.com/items?itemName=pomdtr.excalidraw-editor
2. open the file (eg.`docs/Diagrams/system.excalidraw`)
