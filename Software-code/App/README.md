# Slipstream Flutter Client

Dashboard UI foundation for the Slipstream platform.

## Setup

```bash
cd Software-code/App
flutter pub get
```

## Generate gRPC client

```bash
./tool/gen_protos.sh
```

## Run

```bash
flutter run
```

## Features (Sprint 3)

- Dark mode theme
- Localhost gRPC client
- Calibration wizard flow
- Live connection indicator
- Real‑time telemetry stream view
- E‑Stop UI button
