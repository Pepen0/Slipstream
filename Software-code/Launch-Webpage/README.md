# Slipstream Launch Webpage

This project is a compact 2-DOF motion simulator webpage built with React, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm (or bun)

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd <YOUR_PROJECT_NAME>
   ```

2. Install dependencies:
   ```sh
   npm install
   # or
   bun install
   ```

3. Start the development server:
   ```sh
   npm run dev
   # or
   bun dev
   ```

## Technologies

- **Vite**: Fast build tool and dev server
- **React**: UI library
- **TypeScript**: Static typing
- **shadcn-ui**: Reusable UI components
- **Tailwind CSS**: Utility-first CSS framework
